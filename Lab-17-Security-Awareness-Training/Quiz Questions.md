# Security Awareness Quiz

## 1. What is phishing?

A) A network protocol

B) A cyberattack that tricks users into revealing sensitive information

C) A firewall

D) An antivirus

Answer: B

## 2. Which password is the strongest?

A) password123

B) admin123

C) S3cur3P@ssw0rd!

D) qwerty

## 3. What should you do if you receive a suspicious email?

A) Click the attachment

B) Reply immediately

C) Verify the sender and report it

D) Ignore company policy

Answer: C

## 5. Which is an example of social engineering?

A) Software update

B) Fake IT support call asking for your password

C) Firewall configuration

D) Disk cleanup
